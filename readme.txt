##Exercise for python
這個專案是一個課堂上的開發課題，我透過名為pygame的library來進行遊戲開發的練習。在我的認知中Python更多利用在資料分析或AI的開發，而利用Python進行遊戲開發並不常見。當我發現有pygame這個library讓我很訝異，它引起我的興趣並讓我想嘗試如何使用它來開發遊戲，在此之前我只使用過Unity開發遊戲，這是一個有趣的挑戰。

##
這個遊戲是仿照我小時候玩過的一個遊戲，我不知道它叫甚麼名字，僅依靠我的印象去製作。

這個遊戲的玩法很簡單，透過滑鼠點擊控制直升機不要撞到障礙物，僅此而已。遊戲中直升機會不斷的往前，同時在沒有任何操作的狀態下會持續的墜落。當滑鼠點擊按住左鍵時，可以使直升機向上升起。透過按住滑鼠按鍵及放開滑鼠按鍵的操作控制直升機的高度來躲避障礙物，當你撞到障礙物時遊戲即結束，遊戲中會記錄你的飛行時間並保留前三名紀錄，你可以嘗試不斷突破自己的紀錄，或是跟朋友來一場競賽，看看誰能獲得更多的秒數。
